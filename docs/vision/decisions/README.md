# Architecture Decision Records (ADRs)

Dated, immutable records of significant technical decisions and findings.
Living reference docs (e.g. [`docs/strategies.md`](../strategies.md)) link here
instead of embedding dated decision journals — so the reference stays current
while the decision history is preserved.
